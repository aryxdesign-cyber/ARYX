Signals are generated algorithmically from live exchange data (Binance public API — real OHLCV candles, with CoinGecko as fallback; GeckoTerminal for pasted contract addresses / DEX tokens) using a curated set of technical indicators (Supertrend, EMA 12/21/26/50, ADX regime filter, RSI, Bollinger ±2σ, MACD, Stochastic, MFI, OBV, Hull MA Trend Speed, ROC, volume, Parabolic SAR, Donchian channel breakout, Bollinger-Keltner squeeze, Ichimoku baseline, ATR-stretch guard) with volume-profile POC and variance-ratio market-character context plus candlestick charts, RSI/OBV divergence detection, a watchlist with signal-flip alerts, and a backtest that reports each signal's measured historical hit-rate. Redundant/low-value indicators have been intentionally removed to reduce false consensus. This is an analytical tool, not financial advice. Indicators describe past and present behavior — they do not predict the future. Backtested win-rates are in-sample and optimistic versus live results. Crypto is volatile; never trade more than you can afford to lose, and do your own research.
